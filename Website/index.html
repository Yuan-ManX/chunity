Chunity : ChucK for Unity

version: 1.4.x.x (numchucks)
(up): chuck

Chunity is a plugin for using the ChucK programming language inside the Unity game engine!

Tutorials

You may wish to follow several tutorials to download and get your feet wet with using Chunity. You can also look through the Chunity Boilerplate Project to see more examples of how to use this plugin.
